# 失败修复提示词

## 变脸

补充：

```text
严格保持同一张脸、同样的眼形、鼻子、嘴唇和发型，不要发生脸部变形。
```

## 服装漂移

补充：

```text
严格保持参考图里的黑银服装，不要换装，不要改颜色。
```

## 运动无力

把模糊动作改成节拍动作：

```text
第一个拍点她转头看镜头，第二个拍点她睁眼，第三个拍点镜头快速推进。
```

## 手部崩坏

尽量避免完整手部表演，或者把手做成抽象插入：

```text
优雅的双手部分隐藏在光束中，手指不完全露出，带运动模糊，电影感插入镜头。
```

## 光线太暗

补充：

```text
脸部清晰可见，阴影受控，轮廓光明确，银色边缘高光，避免出现脏黑区域。
```
