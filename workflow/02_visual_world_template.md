# 视觉世界模板

## 一句话

这支 MV 看起来像：

```text
<角色> 在 <世界> 中表演，使用 <光线>、<颜色> 和 <镜头个性>。
```

## 镜头个性

选 2 到 3 个：

- 慢推
- 手持感
- 环绕
- 快速摇移
- 突然拉近
- 静态时尚肖像
- 快速蒙太奇
- 低机位力量镜头

## 避免

- 风格漂移
- 随机地点
- 角色身份变化
- 过于复杂的道具
- 发灰发脏的光线
